## Goal

Delete the LSP diagnostics renderer and strip all diagnostics-block handling from the hashline_edit renderer.

## Owner

backend

## Time

20m

## Difficulty

medium

## Files

- `src/tunacode/ui/renderers/tools/diagnostics.py`
- `src/tunacode/ui/renderers/tools/hashline_edit.py`

## Context

`diagnostics.py` is a dedicated renderer for LSP diagnostics blocks (`<file_diagnostics>`). It is dynamically imported only by the hashline_edit renderer. Since LSP is gone, no diagnostics blocks will ever be produced, so both the renderer and the parsing logic in the hashline_edit renderer should be removed to eliminate dead code.

## Verification

```bash
grep -rn "diagnostics_block\|extract_diagnostics_from_result\|render_diagnostics_inline\|parse_diagnostics_block" src/tunacode/ui/renderers/
```

Should return zero matches.

## Edits

1. Delete `src/tunacode/ui/renderers/tools/diagnostics.py`

2. In `src/tunacode/ui/renderers/tools/hashline_edit.py`:

   a. Remove the `diagnostics_block` field from `EditDiffData`:
      ```python
      diagnostics_block: str | None = None
      ```

   b. In `parse_result`, remove the import and extraction logic:
      ```python
      # Extract diagnostics block before parsing diff
      from tunacode.ui.renderers.tools.diagnostics import extract_diagnostics_from_result

      result_clean, diagnostics_block = extract_diagnostics_from_result(result)
      ```
      Also remove `diagnostics_block=diagnostics_block` from the `EditDiffData(...)` constructor call.

   c. In `render_result`, remove the entire conditional diagnostics-zone block (currently around lines 397-412):
      ```python
      # Add diagnostics zone if present
      if data.diagnostics_block:
          from tunacode.ui.renderers.tools.diagnostics import (
              parse_diagnostics_block,
              render_diagnostics_inline,
          )

          diag_data = parse_diagnostics_block(data.diagnostics_block)
          if diag_data and diag_data.items:
              content_parts.extend(
                  [
                      Text("\n"),
                      separator,
                      Text("\n"),
                      render_diagnostics_inline(diag_data),
                  ]
              )
      ```

   d. Update the class docstring on `HashlineEditRenderer` to remove "with optional diagnostics zone".
